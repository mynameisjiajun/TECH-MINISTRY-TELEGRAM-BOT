====================================================================
GOOGLE SHEETS TEMPLATE FOR CHURCH TECH MINISTRY RENTAL BOT
====================================================================

Create a Google Sheet with TWO sheets:

====================================================================
SHEET 1: "Available Items"
====================================================================

Row 1 (Headers):
ItemID | Item Name | Type | Brand | Model | Quantity | Location

Example Data (Row 2+):
CAB001 | XLR Cable 3m | Cable | Neutrik | NC3MXX | 5 | Shelf A1
CAB002 | SDI Cable 10m | Cable | Canare | L-5CFW | 3 | Shelf A2
MIC001 | Wireless Mic | Microphone | Shure | SM58 | 2 | Cabinet C1
CAM001 | HDMI Camera | Camera | Sony | Alpha | 1 | Cabinet D1

NOTES:
- ItemID must be unique for each item type
- Quantity is the total number of units available
- Location helps users find the item (e.g., "Shelf A1, Drawer 2")
- Type helps categorize items (Cable, Microphone, Camera, etc.)

====================================================================
SHEET 2: "Rental Log"
====================================================================

Row 1 (Headers):
Borrower Name | Telegram Username | User ID | Item ID | Item Name | Rental Start Date | Expected Return Date | Actual Return Date | Status | Pickup Photo | Return Photo

Example Data (Auto-generated by bot):
John Doe | @johndoe | 123456789 | CAB001 | XLR Cable 3m | 2024-01-15 10:30:00 | 2024-01-18 | | ACTIVE | https://api.telegram.org/file/bot.../photo.jpg |
Jane Smith | @janesmith | 987654321 | MIC001 | Wireless Mic | 2024-01-14 09:00:00 | 2024-01-17 | 2024-01-16 14:30:00 | RETURNED | https://api.telegram.org/file/bot.../photo.jpg | https://api.telegram.org/file/bot.../photo.jpg

NOTES:
- This sheet is automatically filled by the bot
- DO NOT manually edit unless necessary
- Status can be: ACTIVE or RETURNED
- User ID is used for sending reminders
- Photo URLs are from Telegram's servers

====================================================================
COLUMN EXPLANATIONS
====================================================================

Available Items Sheet:
- ItemID: Unique identifier (e.g., CAB001, MIC001)
- Item Name: Descriptive name of the item
- Type: Category (Cable, Microphone, Camera, etc.)
- Brand: Manufacturer brand
- Model: Specific model number
- Quantity: Total units available
- Location: Physical location in storage

Rental Log Sheet:
- Borrower Name: User's first and last name from Telegram
- Telegram Username: @username or ID:number format
- User ID: Telegram user ID (for reminders)
- Item ID: Reference to item from Available Items sheet
- Item Name: Copy of item name for easy reference
- Rental Start Date: When item was borrowed (YYYY-MM-DD HH:MM:SS)
- Expected Return Date: When item should be returned (YYYY-MM-DD)
- Actual Return Date: When item was actually returned (YYYY-MM-DD HH:MM:SS)
- Status: ACTIVE (currently borrowed) or RETURNED (returned)
- Pickup Photo: Telegram photo URL of item pickup
- Return Photo: Telegram photo URL of item return

====================================================================
TIPS
====================================================================

1. Use consistent ItemID naming conventions:
   - CAB### for cables
   - MIC### for microphones
   - CAM### for cameras
   - LIGHT### for lighting equipment
   etc.

2. Keep location descriptions clear and specific:
   - "Shelf A1, Top Drawer"
   - "Cabinet C2, Middle Section"
   - "Main Stage Equipment Rack"

3. Regularly review the Rental Log sheet to:
   - Track popular items
   - Identify items that are overdue
   - Monitor usage patterns

4. You can add conditional formatting to highlight:
   - ACTIVE rentals (yellow background)
   - RETURNED rentals (green background)
   - Items past return date (red background)
